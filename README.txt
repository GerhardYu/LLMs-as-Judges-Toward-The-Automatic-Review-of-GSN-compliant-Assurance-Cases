# LLMs as Judges: Toward The Automatic Review of GSN-compliant Assurance Cases

## Overview
This repository contains the resources created for the research work on LLMs as Judges: Toward The Automatic Review of GSN-compliant Assurance Cases.

## Resources Available 

### Fleiss Kappa
**Description** - Contains the code for solving the Fleiss kappa, which contains the fleiss kappa of the following criterions: 
**Zero-Shot:** Argument Comprehension, Well-Formedness (Syntax), Expressive Sufficiency, Argument Criticism and Defeat.
**CoT Zero-Shot:** Argument Comprehension, Well-Formedness (Syntax), Expressive Sufficiency, Argument Criticism and Defeat.
**CoT One-Shot:** Argument Comprehension, Well-Formedness (Syntax), Expressive Sufficiency, Argument Criticism and Defeat.

### Good Thing and Bad Thing
**Description** - A simple taxonomy that shows what the LLMs capabilities and weaknesses are when reviewing assurance cases.

### LLM Review
**Description** - Contains the reviews generated by the LLMs. The LLMs used are as follows: **GPT-4o, GPT 4.1, DeepSeek R1, Gemini 2.0 Flash**.

### Structured Prose
**Description** - Contains the structured prose used for the research project. The structured prose used are as follows: 
**Baidu Apollo:** Adapted from Odu et al. (2025) work on **LLM-Based Safety Case Generation for Baidu Apollo: Are We There Yet?**
**Lane Management System (LMS):** Adapted from Sivakumar et al. (2024) work on **Prompting GPT-4 to support automatic safety case generation**.
**Generic Patient-Controlled Analgesia (GPCA):** Adapted from Lin et al. (2017) work on **Support for safety case generation via model transformation**.
**IM Software:**: Adapted from Xu et al. (2017) work on **A Layered Argument Strategy For Software Security Case Development**.
**Level 4 Automated Driving**: Adapted from Kodama et al. (2024) work on **A Case Study of Continuous Assurance Argument for Level 4 Automated Driving**.

### Prompt Template
**Description** - Contains the different kinds of templates used for the research project. The prompt template contains the following: Zero Shot CoT template, One Shot CoT template, Zero-Shot template, CoT Prompt template.

### Classification of Results
**Description** - Contains the manual assessment done on the LLM-generated reviews.

### Replication Code
**Description** - Contains the replication code for replicating the research.

### Precision, Recall, F1 Score
**Description** - Contains the replication code for the solving the Precision, Recall, and F1 score.
**Precision,Recall,F1Score.ipynb** - Replication code for the Precision, Recall, and F1 Score

### Cohen Kappa
**Description** - Contains the two rater's scoring of the LLM-reviewed assurance case while also including the replication package for solving the Cohen Kappa. The files included are as follows:
**rater1.csv** - Contains the first rater's rating.
**rater2.csv** - Contains the second rater's rating.
**kappa.ipynb** - Contains the replication code for solving the Cohen Kappa.
